# Trader Behavior vs. Bitcoin Market Sentiment

This project analyzes how trader performance varies across different Bitcoin market sentiment phases using historical execution data and the Fear & Greed Index.

# Objective

To uncover patterns between trader behavior and market sentiment, and deliver insights that can guide smarter trading strategies.

# Files

- `analysis_script.py`: Main script for merging, analyzing, and visualizing data
- `sentiment_trader_summary.csv`: Aggregated daily performance by sentiment
- `visualizations/`: Contains charts showing sentiment vs. PnL, win rate, and trade volume

# How to Run

```bash
python analysis_script.py
